# nadlan-tools — כלים חינמיים לשיווק (ראם / כלי נדל"ן חכם)

אוסף מחשבוני נדל"ן חינמיים, **100% צד-לקוח** (HTML/CSS/JS טהור) — אפס שרת, אפס API, אפס בינה מלאכותית. עלות הרצה = אפס לכל מספר משתמשים. נועדו כ-Lead Magnets שמובילים לאפליקציה המלאה ב-https://re-ai-tool.vercel.app

## כלים
- `index.html` — עמוד hub שמרכז את כל הכלים.
- `purchase-tax/` — מחשבון מס רכישה (מדרגות מאומתות יוני 2026, מוקפא 16.1.2024–15.1.2028).
- `rental-tax/` — מדריך אינטראקטיבי למס הכנסה על שכירות (3 מסלולים + מס יסף).
- `yield/` — מחשבון תשואת שכירות (ברוטו/נטו/החזר השקעה).
- `assets/` — לוגו ונכסים משותפים.

## פרסום (GitHub Pages)
1. `gh auth login` (פעם אחת).
2. ליצור repo ציבורי `nadlan-tools` ולדחוף.
3. Settings → Pages → Branch: `main` → `/root`.
4. הכלים יהיו זמינים ב-`https://<user>.github.io/nadlan-tools/` (או דומיין מותג).

## עקרון ברזל
נתוני מס/חוק — רק מאומתים ממקור רשמי. אסור להמציא מספרים. ראה `tax-constants` וההערות בכל קובץ.
